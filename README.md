## âš™ï¸ Configuration

The application now includes a comprehensive configuration system:

### Settings Menu
Access via option 26 in the main menu:
- **Auto Update**: Enable/disable automatic update checking
- **Save Logs**: Control logging to files
- **Max Threads**: Configure thread limits for performance
- **Timeout**: Set network timeout values

### Configuration File
Settings are stored in `apkaless_config.ini`:
```ini
[DEFAULT]
log_level = INFO
auto_update = True
save_logs = True
max_threads = 50
timeout = 30

[API_KEYS]
opencage_key = your_key_here
ipinfo_key = your_key_here
```
